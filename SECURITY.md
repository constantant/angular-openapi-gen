# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| `@constantant/openapi-resource-gen` 1.x (latest) | ✅ |
| older releases | ❌ |

Only the latest published `1.x` release receives security fixes. Please upgrade
before reporting an issue against an older version.

## Reporting a vulnerability

**Please do not open a public issue for security vulnerabilities.**

Report privately using either:

1. **GitHub** — *Security* tab → *Report a vulnerability* (private advisory), or
2. **Email** — **constantant1@gmail.com**

Please include:

- a description of the issue and its impact,
- steps to reproduce (a minimal spec or `nx g` invocation if relevant),
- affected version(s).

You can expect an initial response within **5 business days**. Once a fix is
ready we will coordinate a release and credit you in the advisory unless you
prefer to remain anonymous.
